# Peak Keyboard & Mouse Shortcuts

While Peak is designed for a mouse-heavy workflow, many common operations have keyboard shortcuts.

## 1. Text Editing

- Ctrl-C            Copy selection to clipboard (Snarf)
- Ctrl-X            Cut selection to clipboard (Cut)
- Ctrl-V            Paste from clipboard (Paste)
- Ctrl-Z            Undo last change (Undo)
- Ctrl-Y            Redo last undone change (Redo)
- Ctrl-Shift-Z      Redo last undone change (Redo)
- Ctrl-U            Delete the entire current line
- Ctrl-W            Delete the word before the cursor
- Backspace         Delete character before cursor
- Delete            Delete character at cursor
- Tab               Insert a tab character
- Enter             Insert a newline
- Esc               Cancel selection or select newly typed text

## 2. Navigation

- Arrow             Move cursor Up, Down, Left, Right
- Ctrl-Left/Right   Move cursor by word
- Home              Move cursor to the beginning of the line
- End               Move cursor to the end of the line
- PgUp/PgDn         Move cursor and scroll up/down by one screen

## 3. Search and Commands

- Ctrl-F            Execute Look on the currently selected text (Look)

## 4. Mouse Shortcuts

Peak uses a three-button mouse model:

- Button 1 (Left)   Select text. Drag on vertical scrollbar to scroll.
- Button 2 (Right)  Plumb (Open file/path or search for the word under the cursor)
- Button 3 (Middle) Execute (Run the word under the cursor as a command)

### Scrollbar Interaction (Left Gutter)

- Left Click        Scroll Up (distance from top determines amount)
- Right Click       Scroll Down (distance from top determines amount)
- Middle Click      Jump to the corresponding position in the file
