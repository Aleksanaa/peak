Peak is a TUI text editor inspired by the Plan 9 Acme editor. It follows the philosophy that everything is editable text and text is a command.

Documentation Index:

- Basics.md           : Interaction model and layout.
- Commands.md         : Built-in and shell commands.
- Shortcuts.md        : Navigation and editing keyboard shortcuts.
- StructuralRegex.md  : Advanced text manipulation using Edit.

Usage:

Right-click on any of the filenames above to open them in Peak.
Middle-click commands like Get or Put in the window tag to interact with them.
