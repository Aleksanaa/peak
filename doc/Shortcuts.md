Peak Keyboard Shortcuts

While Peak is designed for a mouse-heavy workflow, many common operations have keyboard shortcuts.

1. Text Editing

- Ctrl-C: Copy selection to clipboard (Snarf).
- Ctrl-X: Cut selection to clipboard.
- Ctrl-V: Paste from clipboard.
- Ctrl-Z: Undo.
- Ctrl-Y or Ctrl-Shift-Z: Redo.
- Ctrl-U: Delete the entire current line.
- Ctrl-W: Delete the word before the cursor.
- Backspace: Delete character before cursor.
- Delete: Delete character at cursor.
- Tab: Insert a tab character.
- Enter: Insert a newline.

2. Navigation

- Arrows: Move cursor Up, Down, Left, Right.
- Ctrl-Left / Ctrl-Right: Move cursor by word.
- Home: Move cursor to the beginning of the line.
- End: Move cursor to the end of the line.
- PgUp / PgDn: Move cursor and scroll up/down by one screen.

3. Search and Commands

- Ctrl-F: Execute Look on the currently selected text. If no text is selected, it searches for the word under the cursor.

4. Mouse-Keyboard Interaction

- Shift + Mouse Click: (Standard terminal behavior usually applies, but Peak handles its own selection logic via Button 1).
- Ctrl + Mouse Click: Reserved for future use or system-specific behavior.
